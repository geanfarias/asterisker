# Asterisk\*r

This is a simple, single page Flask website that turns non-special characters into asterisks. Here is a example, literally:

# \*\*\*\*\*\*\*\*\\\*\*

\*\*\*\* \*\* \* \*\*\*\*\*\*, \*\*\*\*\*\* \*\*\*\* \*\*\*\*\* \*\*\*\*\*\*\* \*\*\*\* \*\*\*\*\* \*\*\*-\*\*\*\*\*\*\* \*\*\*\*\*\*\*\*\*\* \*\*\*\* \*\*\*\*\*\*\*\*\*. \*\*\*\* \*\* \* \*\*\*\*\*\*\*, \*\*\*\*\*\*\*\*\*:
